Cuando un usuario ingresa 3 veces consecutivamente mal la contraseña, esta se bloquea.
En este caso de uso se describe cómo debe ser desbloqueada.
Se quita la marca en la lista de usuarios que indica que el mismo está bloqueado.

03-Desbloquear Contraseña Constraint
Constraint Type Status Detail
El usuario debe contar con un código alfanumérico de 6 caracteres Pre-condition Approved El usuario debe haber recibido un correo con instrucciones sobre cómo reiniciar la contraseña. el mismo contará con un código para el reinicio de contraseña (Ver: *Generar Código de Contraseña*)
Usuario puede usar el sistema Post-condition Approved
