52-Correo de Reiniciar Contraseña Scenarios
Scenario Type Detail
Flujo Básico Basic Path Se enviará un correo con las siguientes características:

- Receptor: El usuario que haya pedido desbloquear contraseña
- Asunto: "Reinicio de Contraseña".
- Contenido: Deberá informar que el usuario está bloqueado y no podrá ingresar nuevamente. Para volver a ingresar deberá ingresar a la dirección referenciada, ingresar el código que se adjunta y una nueva contraseña, así como la repetición de la misma.
- Referencia: Link a pantalla de reinicio de contraseña. También se adjuntará un código de 6 caracteres, alfanumérico. Ver *Generar Código de Contraseña*
