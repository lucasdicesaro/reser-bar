Al seleccionar un restaurante se podrá elegir una opción comparar. Luego de encontrar el segundo restaurante, se podrá elegir una opción que permita compararlo con el previamente seleccionado. Se visualizará la información de ambos restaurantes, así como el perfil de los mismos, que se encuentre configurada para cada uno de ellos.

NOTA: Se recomienda que se visualice en una pantalla con una tabla de doble entrada, siendo cada columna un restaurante que se está comparando y cada fiila un distinto item de comparación
