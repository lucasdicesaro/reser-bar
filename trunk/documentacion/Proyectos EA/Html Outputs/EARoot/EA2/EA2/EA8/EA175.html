Este caso de uso le permitirá a un usuario con permisos, Admitir nuevos Administradores para el restaurante.
Este caso de uso permitirá a un usuario aceptar a otro que lo haya requerido, así como invitar a nuevos usuarios.

15-Admitir nuevos Administradores Scenarios
Scenario Type Detail
Flujo Básico Basic Path Usuario: Ingresa a la Administración de Administradores.
Sistema: Visualiza "Peticiones pendientes de Aceptación"
Usuario: Selecciona una petición de las pendientes.
Sistema: Visualiza los datos del usuario que realizó la petición.
Sistema: Visualiza el comentario que ingresó el usuario requirente.
Usuario: Acepta la petición.
Sistema: Se quita la petición del listado de "Peticiones pendientes de Aceptación".
Sistema: Envia correo al usuario requirente informándole la novededad. *Correo Novedad en Solicitud*
Invitación Simple Usuario: Ingresa a la Administración de Administradores.
Usuario: Selecciona Invitar usuario para ser Administrador del Restaurante.
Sistema: Pide que se ingrese la dirección de correo electrónico del nuevo usuario.
Sistema: Pide que se definan los permisos del usuario que se está invitando.
Usuario: Completa la dirección y permisos asignados al usuario.
Sistema: Pide confirmación
Sitema: Envía correo electrónico al nuevo usuario. *Correo Invitación de Administrador*
Rechaza peitción Alternate Usuario: Rechaza la petición.
Sistema: Se quita la petición del listado de "Peticiones pendientes de Aceptación".
Sistema: Envia correo al usuario requirente informándole la novededad. *Correo Novedad en Solicitud*
