13-Registrar usuario Administrador de Restaurante Scenarios
Scenario Type Detail
Flujo Básico Basic Path Usuario: Selecciona Registración como Administrador de Restaurante.
Usuario: Se registra como Usuario
Sistema: Le pide al usuario que identifique el Restaurante del que quiere ser administrador. Para ello provee el las herramientas de Búsqueda de restaurante.
Usuario:Elige el restaurante que desea administar. (Caso contrario Flujo simple: Restaurante inexistente)
Usuario: Ingresa un comentario.
Sistema: Envia correo al usuario con permiso de autorizar nuevos administradores para el restaurante, del restaurante solicitado.
Sistema: Informa al usuario que deberá esperar una confirmación del Administrador del restaurante.
Sistema: Agrega la solicitud a las solicitudes pendientes de aceptación de dicho Administrador.
Restaurante inexistente Simple Usuario: Selecciona que el restaurante aún no existe en el sistema.
Sistema: Pide confirmación que se desea ingresar un nuevo restaurante al sistema.
Usuario: Acepta.
