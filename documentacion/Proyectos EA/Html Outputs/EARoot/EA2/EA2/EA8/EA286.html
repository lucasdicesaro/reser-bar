Este caso de uso le permitirá a un usuario con permisos, Admitir nuevos Administradores para el restaurante.

Se acepta que un usuario registrado administre el restaurante. Para ello se le debe asignar el mínimo rol para que tenga permisos de administración.
Al admitir nuevo administrador, se asocia al usuario con un rol.
