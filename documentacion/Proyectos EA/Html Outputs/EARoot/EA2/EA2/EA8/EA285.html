Al aceptar una invitación para administrar un restaurante, primero se elige una contraseña para activar la cuenta del usuario y luego se puede continuar con la edición de sus datos.
Al ingresar al sistema, ya tiene asignado el rol necesario para administrar el restaurante y no requiere aceptación de ningún usuario para comenzar a utilizar el sistema.
