Al seleccionar un menú se podrá elegir una opción comparar. Luego de encontrar el segundo menú, se podrá elegir una opción que permita compararlo con el previamente seleccionado. Se visualizará la información de ambos menúes que se encuentre configurada para cada uno de ellos.

NOTA: Se recomienda que se visualice en una pantalla con una tabla de doble entrada, siendo cada columna un menú que se está comparando y cada fiila un distinto item de comparación
