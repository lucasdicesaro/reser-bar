Este caso de uso es una generalización de las posibles estadísticas que se pueden visualizar.

En muchos casos se podrá elegir el período de tiempo para la misma.
Por ejemplo menú más elegido "de la semana", etc.

La granuladida posible es:
- Día
- Semana
- Mes
- Año
- Histórico
