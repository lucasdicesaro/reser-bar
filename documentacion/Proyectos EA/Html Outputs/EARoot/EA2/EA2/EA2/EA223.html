Se enviará un correo electrónico, cuando se algún usuario ingrese una nueva reserva. Se enviará el correo a todos los usuarios que estén ingresados como invitados al mismo.

49-Correo de Invitación a Evento Scenarios
Scenario Type Detail
Flujo Básico Basic Path Se enviará un correo con las siguientes características:

- Receptor: El usuario Invitado
- Asunto: "Invitación a Evento:" <NombreEvento>, de <AliasUsuarioComesal> .
- Contenido: El usuario <Alias> (<Nombre> <Apellido>) te ha invitado a <NombreEvento>. Para aceptar su invitación ingresa a <PantallaEvento>, desde donde también podrás elegir tu menú. El evento se realizará en <NombreRestaurante>, ubicado en las calles <DirecciónRestaurante>.
Mensaje personal:
<MensajePersonal>
- Referencia: Link a pantalla del evento (en <PantallaEvento>. Asimismo se adjuntarán los datos ingresados entre "<" y ">" .
En el campo <MensajePersonal> se adjuntará el mensaje configurado para todos los usuarios, en caso de haberse personalizado el mismo se enviará la perosnalización.
