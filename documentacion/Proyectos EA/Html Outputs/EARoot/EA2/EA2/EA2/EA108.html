46-Correo Invitación de Administrador Scenarios
Scenario Type Detail
Flujo Básico Basic Path Se enviará un correo con las siguientes características:

- Receptor: Usuario que está siendo invitado. El correo ha sido ingresado por el Administrador qeu lo está invitando.
- Asunto: "Inivtación para administrar restaurante desde ReserBar".
- Contenido: El restaurante <NombreRestaurante> te ha invitado a ser Administrador del mismo. Para aceptar la invitación ingresa a <RegistraciónUsuario>
- Referencia: Link a pantalla de Registración de Usuario. También se adjuntará el nombre del Restaurante donde figura "<" y ">" .
