25-Invitar a Evento Scenarios
Scenario Type Detail
Flujo Básico Basic Path Usuario: Ingresa un mensaje para mandarle a todos los usuarios que serán invitados.

Los siguientes pasos se ejecutarán por cada usuarios que quiera ser invitado al evento.

Usuario: Selecciona la opción Agregar nuevo invitado.
Sistema: Le da la opción de agregar la invitación de las siguientes maneras:
a. Ingresando el correo electrónico del usuario. (Simple: Ingresar Correo)
b. Ingresando a la aplicación de Búsqueda de usuarios existentes. (Simple: Búsqueda de usuario)
Usuario: Elige el usuario a invitar.
Usuario: Puede personalizar para este uario el mensaje.
Ingresar Correo Simple Sistema: Disponibilizará un campo para ingresar el correo electrónico y un campo donde se informará el Alias de usuario.
Sistema: Opcionalmente permitirá ver más información sobre el usuario invitado. Se podrá ver:
a. Nombre y Apellido
b. Alias
c. Correo Electrónico
d. Foto de Perfil
Usuario: Ingresará el correo electrónico del usuario
Sistema: Permitirá Aceptar el correo ingresado o ver datos del usuario a invitar (para visualizar los datos mencionados previo a ingresar al usuario).
Usuario: Acepta el Correo Electrónico ingresado
Sistema: Agrega al usuario al listado de invitados
Búsqueda de usuario Simple Sistema: Disponibilizará un campo para ingresar el Nombre y Apellido del usuario invitado, o bien el Alias del mismo.
Sistema: A medida que el usuario ingresa datos en este campo irá verificando la existencia de usuarios con estos datos.
Sistema: Llegada la ocación de no existencia de usuarios con Nombre, Apellido o Alias ingresados se informará y no permitirá invitar a este usuario.
Sistema: Cuando queda un solo usuario que cumpla en su Nombre, Apellido o Alias con los datos ingresados, se sugerirá el mismo.
Usuario: Ingresa en el campo disponible el nombre y/o el apellido o el Alias del usuario invitado.
Usuario: Selecciona el usuario invitado.
Sistema: Opcionalmente permitirá ver más información sobre el usuario invitado. Se podrá ver:
a. Nombre y Apellido
b. Alias
c. Correo Electrónico
d. Foto de Perfil
Sistema: Permitirá Aceptar al Usuario o ver datos del usuario a invitar (para visualizar los datos mencionados previo a ingresar al usuario).
Usuario: Acepta el Usuario invitado ingresado
Sistema: Agrega al usuario al listado de invitados
Enviar invitaciones Alternate Una vez que se haya terminado de realizar la reserva se enviará a cada invitado un correo electrónico.
Para cada invitado se enviará un correo con el siguiente formato. Ver *Correo de Invitación a Evento*
