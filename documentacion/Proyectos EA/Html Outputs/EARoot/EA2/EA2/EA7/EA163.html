Comensal puede realizar una reserva. En la misma se indica la cantidad deseada de cada menú y la fecha y hora de reserva deseada.

DUDAS:
Cuando se hace la reserva se tiene que pagar. si se permite a los usuarios elegir sus menúes, cómo se paga?

16-Realizar Reserva Scenarios
Scenario Type Detail
Flujo Básico Basic Path Usuario: Selecciona opción Realizar Reserva.
Usuario: Seleccionar el restaurante donde quiere hacer la reserva. Ver *Seleccionar Restaurante*.
Sistema: Carga el listado de Eventos disponibles para el restaurante seleccionado. Ver *Editar Perfil Restaurante*
Sistema: Carga el listado de menúes disponibles para el restaurante seleccionado. Ver *Editar Perfil Restaurante*
Sistema: Carga el listado de pomociones disponibles para el restaurante seleccionado. Ver *Editar Perfil Restaurante*
Usuario: Selecciona el Evento del listado de eventos disponibles para el restaurante seleccionado. Ver *Seleccionar Evento*
Usuario: Invita al evento a los usuarios que desee. Ver *Invitar a Evento*
Usuario: Elige darle la posibilidad a los usuarios invitados de elegir sus propios menúes o elegir él el mneú para todos. (Únicamente si habrán invitados).
Usuario: Selecciona un menú de los listados por el restaurante. Este paso se puede repetir tantas veces como se quiera. Ver *Elegir Menú*
Usuario: Elige la cantidad requerida para cada menú seleccioando
Sistema: A medida que se indica la cantidad de menúes que se desea para cada tipo de menú, va actualizando el precio de la reserva.
Usuario: Elige el horario de la reserva.
Usuario: Elige Finalizar Reserva.
Sistema: Valida los datos ingresados anteriormente. Ver *Validación de Reserva*
Sistema: Lista los medios de pago disponibles para efectivizar la reserva.
Usuario: Efectiviza la reserva siguiendo los pasos indicados. Ver *Pago de Reserva*
