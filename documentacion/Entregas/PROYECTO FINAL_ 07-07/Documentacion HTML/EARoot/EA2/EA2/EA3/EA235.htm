Este caso de uso permite el ingreso de un usuario al sistema.
Se ingresa usuario y contraseña. Se verifica la existencia del usuario en el listado de usuarios y se verifica, mediante algún sistema de cifrado, que la contraseña ingresada sea válida. Previamente se verifica que la cuenta del usuario no esté bloqueada
