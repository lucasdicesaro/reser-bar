Este caso de uso enviará un correo a un usuario cuyo permiso (para el restaurante) es el de Administrar los Administradores (Aceptar o Quitar el permiso de administrar el restaurante)

Scenario Type Detail
Flujo Básico Basic Path
Se enviará un correo con las siguientes características:

- Receptor: El usuario del administrador seleccionado cuyo permiso sea Administrar los Administradores
- Asunto: "Autorizar nuevo Administrador".
- Contenido: El usuario <Nombre> <Apellido>, con alias: <Alias> ha solicitado ser Administrador del restaurante. Para más información puede contactarse al correo <CorreoUsuario> o al teléfono <TelefonoUsuario>. Para autorizar al usuario ingrese a la aplicación mediante el siguiente link <LinkPantallaAutorizacion>.
- Referencia: Link a pantalla de Autorización de nuevos usuarios administradores. También se adjuntarán los datos ingresados entre "<" y ">" donde se indica y con los datos del usuario Requirente.
