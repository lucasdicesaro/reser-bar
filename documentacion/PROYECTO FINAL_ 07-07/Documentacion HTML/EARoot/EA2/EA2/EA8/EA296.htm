Se podrán editar los datos de perfil ingresados previamente para el restaurante. Algunos de los datos pueden ser los medios de pago disponibles, preferencias de establecimientos (such us: mesas al aire libre, estacionamiento, juegos para chicos,accesos para discapacitados, Aire acondicionado, etc).

Se configurará el tiempo de recambio (tiempo aproximado de duración de la reserva. Por default 75 minutos), Asimismo deberá detallar la cantidad de comensales disponibles para realizar reservas
